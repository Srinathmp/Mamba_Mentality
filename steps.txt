The above works if the public repo has one branch, what if you want to work on some particular branch?
1. Fork the remote repo
2. git clone https://github.com/Srinathmp/Code-Viz.git
3. git pull origin main
4. git remote -v 
   git remote add origin https://github.com/Srinathmp/Code-Viz.git
5. git fetch origin remote_branch:non_existent_local_branch
6. git checkout non_existent_local_branch
